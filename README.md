# AI Resume Screening System

An AI-based Resume Screening System built using Flask and Natural Language Processing (NLP).
The system analyzes resumes, extracts important information, and matches candidates with job requirements.

## Features

* User authentication (Login/Signup)
* Upload resumes
* Resume parsing
* Skill extraction
* AI-based resume matching
* Candidate analysis dashboard
* Database management

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* Natural Language Processing (NLP)
* SQLite Database

## Installation

Clone the repository:


git clone https://github.com/PraYash2522/resume-screening-system.git


Install required libraries:


pip install -r requirements.txt


## Running the Project

Run Flask application:


python app.py

Open browser:


http://127.0.0.1:5000
